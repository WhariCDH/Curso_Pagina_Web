/************************/
/* 🚀 1. ¿QUÉ ES JAVASCRIPT? */
/************************/

👉 JavaScript es un lenguaje de programación que permite
agregar interacción y dinamismo a una página web.

Con JavaScript podemos:

✔ Responder a clics de botones
✔ Validar formularios
✔ Mostrar mensajes
✔ Realizar cálculos
✔ Crear animaciones
✔ Crear juegos
✔ Modificar HTML y CSS dinámicamente

📌 Fue creado en 1995 por Brendan Eich.

📌 Es un lenguaje del lado del cliente,
porque normalmente se ejecuta en el navegador.

/****************************************/
/* ⚠️ CARACTERÍSTICAS IMPORTANTES */
/****************************************/

✔ Distingue mayúsculas y minúsculas

Ejemplo:

let nombre = "Juan";
let Nombre = "Pedro";

👉 Son variables diferentes.

✔ El punto y coma (;) no es obligatorio,
pero se recomienda usarlo.

Ejemplo:

let edad = 20;

/********************************/
/* 📥 2. IMPLEMENTAR JAVASCRIPT */
/********************************/

JavaScript puede agregarse de 3 formas.

/***************************/
/* 1. EN LÍNEA */
/***************************/

<button onclick="alert('Hola');">
    Presionar
</button>

✔ Se coloca directamente dentro de la etiqueta HTML.

⚠️ Poco recomendado.

/***************************/
/* 2. INTERNO */
/***************************/

<script>

    alert("Hola Mundo");

</script>

✔ Se coloca dentro del HTML.

/***************************/
/* 3. EXTERNO (RECOMENDADO) */
/***************************/

<script src="script.js"></script>

✔ Código separado del HTML.
✔ Más ordenado.
✔ Más profesional.

/********************************/
/* 📊 3. TIPOS DE DATOS */
/********************************/

JavaScript maneja varios tipos de datos.

let nombre = "Whari";      // Texto (String)

let edad = 20;            // Número (Number)

let activo = true;        // Booleano

let nota = 18.5;          // Decimal

let dato = null;          // Vacío

let valor;                // Undefined

/********************************/
/* 📝 4. VARIABLES */
/********************************/

Las variables almacenan información.

Se pueden declarar con:

✔ var
✔ let

Ejemplos:

var curso = "Web";

let ciclo = 5;

📌 Actualmente se recomienda usar:

let

porque es más segura.

/********************************/
/* 🔒 5. CONSTANTES */
/********************************/

Una constante almacena un valor fijo.

const PI = 3.1416;

⚠️ No puede modificarse.

const PI = 3.1416;

PI = 10;

❌ Error

/********************************/
/* ➕ 6. OPERADORES ARITMÉTICOS */
/********************************/

let a = 10;
let b = 5;

Suma
a + b

Resta
a - b

Multiplicación
a * b

División
a / b

Módulo (residuo)
a % b

Potencia
a ** b

/********************************/
/* ⚖️ 7. OPERADORES RELACIONALES */
/********************************/

Se usan para comparar valores.

==   Igual

!=   Diferente

>    Mayor que

<    Menor que

>=   Mayor o igual

<=   Menor o igual

Ejemplo:

10 > 5

Resultado:

true

/******************************/
/* 🔗 8. OPERADORES LÓGICOS */
/******************************/

&&   AND (Y)

||   OR (O)

!    NOT (NO)

Ejemplo:

edad >= 18 && edad <= 60

✔ Ambas condiciones deben cumplirse.

/********************************/
/* 📝 9. OPERADORES DE ASIGNACIÓN */
/********************************/

Asignación básica:

let numero = 10;

numero = numero + 5;

Forma corta:

numero += 5;

Otros:

+=

-=

*=

/=

%=

Ejemplo:

let puntos = 100;

puntos += 50;

Resultado:

150

/********************************/
/* 🖱️ 10. EVENTOS */
/********************************/

Los eventos son acciones realizadas por el usuario.

Ejemplos:

✔ Clic en botón
✔ Escribir texto
✔ Mover el mouse
✔ Cargar página

/************************/
/* EVENTO onclick */
/************************/

<button onclick="saludar()">
    Saludar
</button>

👉 Se ejecuta al hacer clic.

/************************/
/* EVENTO onchange */
/************************/

<select onchange="cambiar()">
</select>

👉 Se ejecuta cuando cambia una opción.

/************************/
/* EVENTO onmouseover */
/************************/

<div onmouseover="mensaje()">
</div>

👉 Se ejecuta cuando el mouse pasa encima.

/************************/
/* EVENTO onload */
/************************/

<body onload="inicio()">

👉 Se ejecuta al cargar la página.

/********************************/
/* 💬 11. MENSAJES BÁSICOS */
/********************************/

/* Mostrar mensaje */

alert("Bienvenido");

alert("Bienvenido " + nombre);

</script>
