/***********************/
/* 🎨 1. ¿QUÉ ES CSS? */
/***********************/
👉 CSS (Cascading Style Sheets) sirve para:

✔ Dar diseño y estilo a una página web
✔ Separar:

HTML → estructura
CSS → apariencia

/********************************/
/* 🧩 2. ¿DÓNDE SE ESCRIBE CSS? */
/********************************/
✔ Forma correcta (LA QUE DEBES USAR)

Archivo externo:
<link rel="stylesheet" href="estilos.css">

✔ Otras formas (menos usadas)

🔹 Dentro del HTML:
<style>
h1 { color: red; }
</style>

🔹 En línea:
<h1 style="color:red;">Hola</h1>

/**************************************************/
/* 🧠 3. CÓMO SE DEFINE UN ESTILO (TU IMAGEN 🔥) */
/**************************************************/
La estructura es:

selector {
    propiedad: valor;
}

💻 Ejemplo de tu imagen:

h1 {
    color: #FF0000;
    background: #00FF00;
}

🔍 PARTES IMPORTANTES
👉 Te lo explico como en la imagen:

Selector → h1
👉 A quién se aplica

Propiedad → color
👉 Qué quieres cambiar

Valor → #FF0000
👉 Cómo quieres que se vea

:` → separa propiedad y valor
; → separa cada regla

/*****************************/
/* 🎯 4. ¿CÓMO FUNCIONA CSS? */
/*****************************/
✔ Puedes aplicar estilos a muchos elementos
✔ Los estilos se pueden sobrescribir
✔ El último estilo tiene prioridad

⚠️ IMPORTANTE (PRIORIDAD)
color: red !important;

👉 Obliga a que ese estilo gane 🔥

/*****************************/
/* 🧩 5. TIPOS DE SELECTORES */
/*****************************/

/* 🔹 1. Selector de elemento */
h1 {
    color: blue;
}

✔ Aplica a todos los <h1>

/* 🔹 2. Selector por ID */
#titulo {
    color: red;
}

✔ Solo a:

<h1 id="titulo">

⚠️ Se usa poco (solo casos especiales)

/* 🔹 3. Selector por CLASE (EL MÁS IMPORTANTE 🔥) */
.texto {
    color: green;
}

✔ Se usa así:

<p class="texto">

👉 ESTE ES EL QUE MÁS VAS A USAR

/***********************************/
/* 🧠 6. BUENAS PRÁCTICAS (CLAVE) */
/***********************************/
✔ Usar class para estilos
✔ Evitar usar id para diseño
✔ Mantener CSS en archivo externo
✔ Ordenar bien los estilos

/*********************************************/
/* ⚠️ 7. COSAS IMPORTANTES QUE DIJO TU PROFE */
/*********************************************/
✔ CSS puede sobrescribirse
✔ El orden importa
✔ Las plantillas pueden afectar tu CSS
✔ Usar !important solo si es necesario

/****************/
🎯 RESUMEN FINAL
/****************/
✔ CSS = diseño
✔ Selector → a quién
✔ Propiedad → qué cambias
✔ Valor → cómo se ve
✔ .class → lo más usado
✔ #id → poco usado
✔ Archivo externo → lo correcto
