🎯 ¿Qué es un formulario?

Un formulario es un conjunto de elementos que permite al usuario ingresar y enviar datos a una página web o servidor.

📌 Ejemplo:

Registrarse
Comprar pasajes
Iniciar sesión

/************************/
/* ESTRUCTURA BÁSICA */
/************************/

<form action="" method="">
</form>

🔹 Atributos importantes:
-action → a dónde se envían los datos
-method → cómo se envían (GET o POST)


⚠️ GET vs POST

GET	                        POST
Datos visibles en la URL	  Datos ocultos
Menos seguro	              Más seguro
Límite de datos	            Permite más datos
No permite archivos	        Permite archivos

👉 Recomendación: usar POST en formularios importantes

/********************************/
/* 1: INPUT (LO MÁS IMPORTANTE) */
/********************************/
La etiqueta <input> crea campos para ingresar datos.

<input type="text">

🔹 TIPOS DE INPUT MÁS USADOS

/* 📝Texto */
<input type="text" name="nombre">

/* 🔒 Contraseña */
<input type="password" name="clave">

/* ☑️ Checkbox (varias opciones) */
<input type="checkbox"> Opción

/* 🔘 Radio (una sola opción) */
<input type="radio" name="sexo"> Hombre
<input type="radio" name="sexo"> Mujer

✔ Solo se puede elegir una
✔ Deben tener el mismo name

/* 📤 Enviar */
<input type="submit" value="Enviar">

/* 🔄 Resetear */
<input type="reset" value="Limpiar">

/***************************/
/* 2: OTROS INPUTS (HTML5) */
/***************************/

<input type="email">
<input type="number">
<input type="date">
<input type="color">
<input type="range">

👉 Mejoran la experiencia del usuario 🔥

/*************************/
/* 2: 🗂️ AGRUPAR CAMPOS */
/*************************/

<fieldset>
  <legend>Datos personales</legend>
</fieldset>

✔ Organiza el formulario
✔ Mejora presentación

/*********************************/
/* 3: 📝 TEXTAREA (texto largo) */
/*********************************/

<textarea rows="5" cols="30"></textarea>

✔ Para comentarios o descripciones

/*************************************/
/* 4: 📋 SELECT (lista desplegable) */
/*************************************/

<select>
  <option>Opción 1</option>
  <option>Opción 2</option>
</select>

🔹 Agrupar opciones

<optgroup label="Grupo 1">
  <option>Opción A</option>
</optgroup>

/*******************************/
🧠 BUENAS PRÁCTICAS (IMPORTANTE)
/*******************************/

✔ Usar <label> para cada campo
✔ Agrupar con <fieldset>
✔ Usar name en inputs (clave para enviar datos)
✔ Elegir bien el type

/******************************************/
/* 🧱 ESTRUCTURA GENERAL DE UN FORMULARIO */
/******************************************/

<form method="post">

  <fieldset>
    <legend>Formulario</legend>

    <label>Nombre:</label>
    <input type="text" name="nombre"><br><br>

    <label>Contraseña:</label>
    <input type="password" name="clave"><br><br>

    <input type="submit" value="Enviar">

  </fieldset>

</form>

/***********************/
🎯 RESUMEN FINAL

✔ <form> → contenedor
✔ <input> → campos
✔ type → tipo de dato
✔ <textarea> → texto largo
✔ <select> → lista
✔ <fieldset> → agrupar
