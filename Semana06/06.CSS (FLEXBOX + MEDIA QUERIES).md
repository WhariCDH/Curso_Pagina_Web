/********************************/
/* 📏 1. UNIDADES VH y VW       */
/********************************/
👉 Son unidades basadas en el tamaño de la pantalla (viewport)

vh → altura de la pantalla  
vw → ancho de la pantalla  

💻 Ejemplo:
height: 100vh; /* ocupa todo el alto */
width: 100vw;  /* ocupa todo el ancho */

⚠️ IMPORTANTE
✔ % depende del elemento padre  
✔ vh/vw depende de la pantalla  

👉 Esto es clave para el examen

/********************************/
/* 📦 2. FLEXBOX (IMPORTANTE 🔥) */
/********************************/
👉 Sistema para ordenar, alinear y distribuir elementos

🧩 Activación:
display: flex;

✔ Se aplica al contenedor  
✔ Los hijos → "flex items"

/*****************************************/
/* 🧭 3. PROPIEDADES DEL CONTENEDOR      */
/*****************************************/

/* 🔁 flex-direction (dirección) */
flex-direction: row;

row → horizontal  
column → vertical  

/* 🔄 flex-wrap (salto de línea) */
flex-wrap: wrap;

nowrap → una sola línea  
wrap → varias líneas  

/* ⚡ flex-flow (atajo) */
flex-flow: row wrap;

/* ↔️ justify-content (horizontal) */
justify-content: center;

flex-start  
center  
space-between  
space-around  
space-evenly  

/* ↕️ align-items (vertical) */
align-items: center;

flex-start  
center  
flex-end  
stretch  

/* 📚 align-content (varias filas) */
✔ Solo funciona si hay varias líneas  

/********************************/
/* 🧱 4. PROPIEDADES DE ITEMS   */
/********************************/

/* 🔢 order */
order: 2;
✔ Cambia el orden visual  

/* 📈 flex-grow */
flex-grow: 1;
✔ Hace que crezca  

/* 📉 flex-shrink */
flex-shrink: 1;
✔ Hace que se reduzca  

/* 📏 flex-basis */
flex-basis: 200px;
✔ Tamaño inicial  

/* 🎯 align-self */
align-self: center;
✔ Alinea un elemento individual  

/*************************************/
/* 📱 5. MEDIA QUERIES (RESPONSIVE) */
/*************************************/
👉 Permiten adaptar la web según el dispositivo

celular 📱  
tablet 📲  
PC 💻  

💻 Sintaxis:
@media (max-width: 600px) {
    body {
        background-color: red;
    }
}

📏 Tamaños comunes:
320px → celular  
768px → tablet  
1200px → PC  

🧠 Tipos:
screen → pantallas  
print → impresión  
all → todos  

🔍 Media features:
width / height  
max-width / min-width  

/********************************/
/* 🎯 EJEMPLO CLAVE             */
/********************************/
@media (max-width: 768px) {
    nav {
        flex-direction: column;
    }
}

👉 El menú cambia en celular 🔥

/********************************/
/* 🧠 IDEAS IMPORTANTES         */
/********************************/
✔ Flexbox = ordenar elementos fácil  
✔ justify-content = horizontal  
✔ align-items = vertical  
✔ vh/vw = pantalla completa  
✔ media queries = responsive
