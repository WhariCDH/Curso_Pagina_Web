/* SEMANA 4: IFRAMES EN HTML5 */
/******************************/
🎯 ¿Qué es un iframe?
La etiqueta <iframe> permite insertar una página web dentro de otra página web.
👉 Es como una “ventana” dentro de tu página.

/*************************/
/* 🧩 ESTRUCTURA BÁSICA */
/*************************/

<iframe src="https://www.ejemplo.com"></iframe>

✔ src → URL de la página que quieres mostrar

📌 ¿Para qué sirve?

✔ Mostrar:

Otra página web
Videos (YouTube)
Mapas (Google Maps)
Documentos

👉 Muy usado en páginas modernas 🔥

/****************************/
/* ⚙️ ATRIBUTOS IMPORTANTES */
/****************************/

/* 📏 Tamaño */

<iframe src="..." width="600" height="400"></iframe>

✔ Define ancho y alto

/* 🚫 Quitar bordes */

frameborder="0"

✔ Elimina el borde del iframe

/* 🔄 Scroll */

scrolling="no"

✔ Oculta barras de desplazamiento

/* 🏷️ Nombre */

name="miIframe"

✔ Permite interactuar con enlaces

/*********************************/
/* 🧠 ATRIBUTO AVANZADO (HTML5) */
/*********************************/

/* 📄 srcdoc */
Permite escribir contenido HTML dentro del iframe:

<iframe srcdoc="<h1>Hola</h1>"></iframe>

👉 No necesita URL externa

/******************/
🧭 CONCEPTO CLAVE

✔ <iframe> crea un contexto independiente
✔ Es como otra página dentro de tu página

/*******************************/
⚠️ DIFERENCIA: IFRAME vs FRAMES

❌ Frames (ANTIGUO)
Dividían la página en varias partes
Están obsoletos en HTML5

✅ Iframe (ACTUAL)
Inserta contenido dentro de una sección
Sí se usa hoy en día

/*********************/
⚠️ COSAS IMPORTANTES

❌ No todas las páginas permiten ser embebidas (seguridad)
❌ Puede afectar rendimiento si abusas

/**********************/
🎯 EJEMPLOS DE USO REAL

✔ Video de YouTube
✔ Mapa de ubicación
✔ Página externa
✔ Vista previa de otra web

/*****************/
🧠 RESUMEN FINAL

✔ <iframe> → incrusta otra página
✔ src → URL
✔ width / height → tamaño
✔ frameborder → borde
✔ scrolling → scroll
✔ srcdoc → contenido interno
