/************************/
/* 🎨 1. ¿QUÉ ES CSS?  */
/************************/

👉 CSS (Cascading Style Sheets) permite definir la apariencia
de una página web.

✔ HTML = estructura
✔ CSS = diseño y presentación

Ejemplos:
- Colores
- Fuentes
- Márgenes
- Bordes
- Distribución de elementos

/********************************/
/* 📚 2. NIVELES DE CSS         */
/********************************/

CSS ha evolucionado en varias versiones:

✔ CSS1
✔ CSS2
✔ CSS2.1
✔ CSS3

Actualmente se utiliza principalmente CSS3.

/********************************/
/* 🌐 3. HTML Y CSS            */
/********************************/

HTML organiza el contenido:

<h1>
<p>
<img>
<table>

CSS mejora su apariencia:

color
background
font-size
margin

✔ HTML = contenido
✔ CSS = diseño

/****************************************/
/* 🏗️ 4. ESTRUCTURA SEMÁNTICA HTML5     */
/****************************************/

Antes se utilizaba principalmente:

<div>

HTML5 incorpora etiquetas semánticas:

<header>
<nav>
<main>
<section>
<article>
<aside>
<footer>
<address>

Ventajas:

✔ Código más ordenado
✔ Mejor posicionamiento SEO
✔ Más fácil de mantener

/********************************/
/* 🧩 5. ESTRUCTURA DE CSS      */
/********************************/

Formato básico:

selector {
    propiedad: valor;
}

Ejemplo:

h1 {
    color: red;
    font-size: 30px;
}

Partes:

Selector → h1
Propiedad → color
Valor → red

/********************************/
/* 🔗 6. FORMAS DE INCLUIR CSS */
/********************************/

/* CSS Embebido */
<p style="color:red;">
Hola Mundo
</p>

⚠️ Poco recomendado

/* CSS Interno */

<style>
p{
    color:red;
}
</style>

⚠️ Solo para pruebas

/* CSS Externo */

<link rel="stylesheet" href="estilos.css">

✔ Forma recomendada

/********************************/
/* 🎯 7. TIPOS DE SELECTORES   */
/********************************/

/* Selector de etiqueta */

h1{
    color:red;
}

/* Selector ID */

#titulo{
    color:blue;
}

✔ Solo un elemento

/* Selector Clase */

.texto{
    color:green;
}

✔ Puede repetirse

/* Selector de atributo */

input[type="text"]{
    border:1px solid black;
}

/* Selector de pseudoclase */

a:hover{
    color:red;
}

✔ Se activa al pasar el mouse

/********************************/
/* 📱 23. MEDIA QUERIES       */
/********************************/

Permiten adaptar la web
a diferentes dispositivos.

Ejemplo:

@media (max-width:768px){

    body{
        background:red;
    }

}

Tamaños comunes:

320px → celular
480px → celular grande
600px → tablet pequeña
768px → tablet
900px → laptop pequeña
1200px → escritorio
