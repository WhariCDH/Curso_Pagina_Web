/*EXPLICACIÓN (PASO A PASO)*/

/*****************************************************/
/*1.🔝 Estructura básica*/

<!DOCTYPE html>
<html>
<head>
<body>

Define que es una página HTML
<head> → configuración
<body> → contenido visible

/*****************************************************/
/*2. 🏷️ Header (Encabezado)*/

<header>
    <h1>Transporte Público Piura</h1>
    <p>Movilízate rápido y seguro</p>
</header>

✔ Aquí va el título y descripción
✔ Es lo primero que ve el usuario

/*****************************************************/
/*3. 🧭 Nav (Menú)*/

<nav>
    <a href="#">Inicio</a> |
    <a href="#">Rutas</a>
</nav>

✔ Sirve para navegar
✔ <a> son enlaces

/*****************************************************/
/*4. 📦 Section + Article*/

<section>
    <article>
        <h2>Rutas Disponibles</h2>
    </article>
</section>

✔ <section> → agrupa contenido
✔ <article> → contenido independiente

/*****************************************************/
/*5. 🖼️ Figure (Imagen)*/

<figure>
    <img src="..." alt="Bus">
    <figcaption>Unidad de transporte</figcaption>
</figure>

✔ Muestra imagen
✔ figcaption describe la imagen

/*****************************************************/
/*6. 📌 Aside (Extra)*/

<aside>
    <h3>Horario</h3>
</aside>

✔ Información adicional
✔ No es lo principal

/*****************************************************/
/*7. 🔚 Footer (Pie de página)*/
<footer>
    <p>© 2026 Transporte Piura</p>
</footer>

✔ Información final
✔ Derechos, contacto, etc.
