/* 🌐 1. ¿Qué es HTML5? */
/*************************/
-Es la última versión de HTML.
-Sirve para crear páginas web.
Permite:
-Mejor estructura (semántica)
-Uso de audio, video e imágenes
-Mejor rendimiento
-Funcionar incluso sin conexión
-Integrarse con CSS3 (diseño) y JavaScript (funcionalidad)

/*******************************************/
/*🧩 2. ¿Cómo se construye una página web?*/
/*******************************************/
Una web tiene 3 partes principales:

-Estructura → HTML
-Apariencia → CSS
-Comportamiento → JavaScript

📌 Ejemplos:

-HTML: textos, imágenes, enlaces
-CSS: colores, tamaños, estilos
-JS: botones, validaciones, animaciones

/*********************************************/
/*🏗️ 3. Componentes de HTML (MUY IMPORTANTE)*/
/*********************************************/
Un elemento HTML tiene esta forma:

<etiqueta atributo="valor">Contenido</etiqueta>

🔹 Partes:
-Etiqueta de apertura: <p>
-Contenido: texto o elementos
-Etiqueta de cierre: </p>
-Atributos: información extra (ej: class, id)

📌 Ejemplo:

<p class="texto">Hola mundo</p>

/**********************************************/
/*🧱 4. Etiquetas semánticas (CLAVE EN HTML5)*/
/**********************************************/
Estas etiquetas indican el significado del contenido:

-<header> → encabezado
-<nav> → menú de navegación
-<section> → sección principal
-<article> → contenido independiente
-<aside> → contenido secundario
-<footer> → pie de página
-<figure> → imagen o gráfico
-<figcaption> → descripción de la imagen

📌 Antes se usaba mucho <div>, ahora se usan estas etiquetas.

/*********************************************/
/*🏛️ 5. Estructura básica de una página HTML*/
/********************************************/
Una página web sigue este esquema:

<html>
  <head>
    <title>Título</title>
  </head>
  <body>
    <header></header>
    <nav></nav>
    <section>
      <article></article>
    </section>
    <footer></footer>
  </body>
</html>

/***************************************/

/***************************************/

✔ Las etiquetas tienen apertura y cierre
✔ Los atributos agregan información
✔ HTML5 usa etiquetas semánticas (más ordenadas)
