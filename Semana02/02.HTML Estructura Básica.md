/* 1. TABLAS (MUY IMPORTANTE) */
/******************************/
Sirven para organizar datos en filas y columnas.

/* Estructura básica */

<table>
  <tr>
    <td>Dato</td>
  </tr>
</table>

🔹 Etiquetas:
<table> → tabla
<tr> → fila
<td> → celda

/* Encabezados */

<th>Ruta</th>

✔ Se usan en la primera fila
✔ Dan título a las columnas

/* Título de tabla */

<caption>Rutas de Transporte</caption>

✔ Describe la tabla
✔ Va justo después de <table>

/* Unir celdas (MUY USADO) */

Horizontal:
<td colspan="2">Texto</td>

Vertical:
<td rowspan="2">Texto</td>

/*Extra (esto casi no lo explican bien)*/
<thead> → encabezado
<tbody> → contenido
<tfoot> → pie
👉 Esto mejora organización (y te suma puntos en prácticas)

/************/
/* 2. LISTA */
/************/

/* Lista ordenada (numerada) */

<ol>
  <li>Subir al bus</li>
  <li>Pagar pasaje</li>
</ol>

/* Lista desordenada (viñetas) */

<ul>
  <li>Ruta A</li>
  <li>Ruta B</li>
</ul>

/* Listas anidadas (IMPORTANTE) */

<ul>
  <li>Rutas
    <ul>
      <li>Centro</li>
      <li>Norte</li>
    </ul>
  </li>
</ul>

👉 Muy útil para menús o categorías

/***************/
/* 3. IMÁGENES */
/***************/

/* Básico */

<img src="bus.jpg" alt="Bus de transporte">

🔹 Atributos:
src → ruta de la imagen
alt → descripción (IMPORTANTE)

🔹 Importante (no siempre lo dicen)
Usa imágenes:
Contenido → <img>
Decoración → CSS

/* Responsive con <picture> */
Sirve para mostrar imágenes según el tamaño de pantalla:

<picture>
  <source media="(max-width: 600px)" srcset="bus-pequeno.jpg">
  <img src="bus.jpg" alt="Bus">
</picture>

👉 Esto es nivel más avanzado 💡

/************/
/* 4. AUDIO */
/************/

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

✔ controls → muestra botones
✔ Puedes poner anuncios, música, etc.

/************/
/* 5. VIDEO */
/************/

<video controls width="300">
  <source src="video.mp4" type="video/mp4">
</video>

✔ Similar al audio
✔ Muy usado en páginas modernas
